In order to get the json configuration of your vless tls configuration follow these steps:

1. Paste your default vless:// style configuration inside v2rayng.
2. Go to routing settings inside v2rayng and turn everything off.
3. Go to v2rayng settings and turn off mux.
4. If you want to test with Fragment turn it on and set your values, if not turn it off.
5. Tap the share button next to your configuration and tap export full configuration to clipboard.
6. Paste it inside the app.
